Kalshi Market Calibration Analysis

Overview
This project analyzes the calibration accuracy of Kalshi prediction markets across different categories. Recent studies sparked debate about whether prediction markets are truly accurate (Clinton & Huang 2024), with platforms like Kalshi defending themselves by claiming strong calibration. 

Architecture - Batch ELT
Kalshi API, Prefect, Snowflake, dbt, Tableau
